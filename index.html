Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • for note above, first make sure you are in the main branch before creating new branch
  • git checkout main: takes you to main branch
  • git add -A: add changed files from our directory to staging area of git
  • git commit -m "insert comment here": to commit changes (m allows to add comment)
  • git pull origin main: make sure branch is up to date before opening pull request
  • git push origin feature/branch-name: pushes all changes through
  • once changes have been pushed navigate to github
